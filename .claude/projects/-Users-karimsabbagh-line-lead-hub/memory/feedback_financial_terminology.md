name: Financial terminology
description: Use "Output Value", "Operating Cost", and "Operating Margin" instead of Revenue/Cost/Profit everywhere
type: feedback
---

Never use "Revenue", "Cost", or "Profit" as labels in financials. The correct terms are:
- Revenue → **Output Value**
- Cost → **Operating Cost**
- Profit → **Operating Margin**
- Profit Margin → **Margin %**

**Why:** User explicitly corrected this — these are the standard terms for the factory context.

**How to apply:** Any new or modified financial display across all pages (Dashboard, TodayUpdates, ThisWeek, Insights, Finances, reports/PDFs) must use these terms.
